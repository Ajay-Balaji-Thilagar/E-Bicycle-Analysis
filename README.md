# E-Bicycle Market Analysis

## Project Overview
This project analyzes E-Bicycle data collected from Amazon and Flipkart.

## Tools Used
- Python
- Jupyter Notebook
- Pandas
- BeautifulSoup
- Requests & RE
- Power BI

## Project Workflow
1. Data Collection/Scrapping
2. Data Cleaning
3. Exploratory Data Analysis
4. Dashboard Creation

## Files
- notebooks/ : Python code
- data/raw/ : Raw scraped datasets
- data/cleaned/ : Cleaned datasets
- powerbi/ : Power BI dashboards
